### Подключение КВ

В = Верхняя рейка
Н = Нижняя рейка

Главный вал:
Кулачки: В-3, Н-1, Н-3, земля
Перемычки: (В-1, Н-4)

Реверсивный вал:
Кулачки: Н-2, Н-3, земля
Перемычки: нет
